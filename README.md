# Inspector Screenshot Utility

**Inspector Screenshot Utility** is a practical Unity Editor tool that lets you capture high-quality screenshots directly from the **Inspector** of any **ScriptableObject**.

Ideal for creating visual previews, documenting assets, or generating accurate images of editor configurations.

---
